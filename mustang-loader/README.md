# 🚗 1967 Ford Mustang Fastback – GSAP Landing Page

A cinematic and immersive landing page inspired by the legendary 1967 Ford Mustang Fastback. The project combines smooth loading animations, a percentage-based preloader, and elegant GSAP transitions to create a premium user experience.

## ✨ Features

* Animated loading screen with percentage counter
* Smooth GSAP timeline animations
* Full-screen hero image
* Premium typography and cinematic design
* Responsive layout
* Sequential text reveal animations
* Modern Vite development setup

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)
* Vite

## 🎬 Animation Flow

1. Loading counter starts from **0%** and increments to **100%**
2. Counter fades out smoothly
3. Overlay slides downward and reveals the page
4. Background image scales into position
5. Main heading animates upward
6. Subheading appears with a smooth reveal effect
